1D regression is limited. Need for many inputs/outputs and also non-linearity can not be modelled with just straight lines.

Here is a simple 1D model (represented in the equation form): 

$$
y = f[x, \phi] = \phi_0 + \phi_1 x
$$

We have two "parameters". Slope $\phi_1$ and intercept on y-axis $\phi_0$.

This shallow network, on the other hand, has 10 parameters with just 1 input and 1 output:

$$
y = f[x, \phi] 
$$

$$
y = \phi_0
+ \phi_1\, a[\theta_{10} + \theta_{11} x] 
+ \phi_2\, a[\theta_{20} + \theta_{21} x] 
+ \phi_3\, a[\theta_{30} + \theta_{31} x]
$$

> Shallow neural networks are functions (family of functions, technically) which uses a combination of other functions.

Notice, we apply the activation function − $a\ [.]$ three times. Most common activation function is ***rectified linear unit*** or more commonly called ***ReLU***:

$$
a[z] = \mathrm{ReLU}[z] =
\begin{cases}
0, & z < 0 \\
z, & z \ge 0
\end{cases}
$$

For positive values, it just returns the input as the output. For negative values of inputs it clips to 0.

The set of a parameters in a small neural network can be given by:

$$
\boldsymbol{\phi} = \{\phi_0, \phi_1, \phi_2, \phi_3, \theta_{10}, \theta_{11}, \theta_{20}, \theta_{21}, \theta_{30}, \theta_{31}\}
$$

The moment we these parameters, they can have different values and hence we are reminded of an important point that:

> A neural network is just family of equations.

Once these values are "settled" or decided on by the process of ***learning***, we can start producing outputs (recall that this is called ***inference***). 

For a training dataset, $\{\mathbf{x}_i, \mathbf{y}_i\}_{i=1}^I$ and a loss function, $L[\boldsymbol{\phi}]$ - we can begin training to learn these 10 parameters.

Taking forward a shallow network with three inputs, each of which gets scaled and added with a bias and then the output is eaten up an activation function:

$$
h_1 = a[\theta_{10} + \theta_{11}x] \\
h_2 = a[\theta_{20} + \theta_{21}x] \\
h_3 = a[\theta_{30} + \theta_{31}x] \\
$$

Outputs of activation functions are ***hidden units***. This gives a rather simpler form of the equation of a simple neural network:

$$
y = \phi_0
+ \phi_1\, h1
+ \phi_2\, h2
+ \phi_3\, h3 
$$

From the plots, we can see the $\phi$ scales the output from the activation. It can:
- flip the curve if $\phi < 0$
- increase the slope if $\phi > 1$
- decrease the slope if $0 < \phi < 1$

Next we sum these curves. Slopes from all three hidden units are being added. Hidden units are active (non-zero) or inactive (zero) in different regions.

We see that a ***piece-wise linear function*** emerges with number of joints as same as the number of hidden units (this might not always be the case, the statement is better qualified with "at most"). Here we have three joints:

The discontinuity in this curve comes from ReLU clips! Hence, we can say that the ReLU brings in the non-linearity.

The $\phi_0$ term will shift (translate) this curve up and down. Down if the value is negative and up if it is positive.

> Each parameter multiplies at source and adds the target.

Here we can see a shallow network with one input layer, one hidden layer (three units or less preferably three neurons), and, one output layer. The moment we add one more hidden layer we get a ***deep neural network***. Number hidden layers is a good estimation of ***model capacity***.


### Universal Approximation Theorem

When we have $D$ hidden units:

$$
h_{d} = a[\theta_{d0} + \theta_{d1}x]
$$

$$
y = \phi_{0} + \sum_{d=1}^{D} \phi_{d}h_{d}
$$

> With enough hidden units, a shallow neural network can describe any continuous function on a compact subset of $\mathbb{R}^D$ to abritary accuracy.
